使用Hexo构建github pages静态博客网站

准备工作

  1. 在github中新建username.github.io仓库（建议勾选上README.md)

  2. 进入Setting-->Pages 点Visit site即可看到网页

  3. Source选择Deploy from a branch

    Branch 选择文件部署仓库的分支的/root

安装Nodejs环境

Node.js 中文网

打开cmd输入node -v 查看是否安装成功

安装git

Git

在任意文件夹用右键 Git Bash Here

安装 Hexo

查看版本 hexo -v （报错的话试试npx hexo -v）

创建项目

安装主题 hexo-theme-yun

修改 Hexo 站点配置文件 _config.yml

后续可按照自己需求修改主题文件夹下的_config.yml（各种美化）（可以在_config.yml，同级目录下新建_config-yun.yml`进行修改）

参考文档主题配置 | Hexo-Theme-Yun

绑定域名（可选）

  1. 购买域名
  2. 建议使用cloudflare代理域名（不需要国内备案）
  4. 进入username.github.io的settings/Pages在Custom domain中填入个人域名
