    Linux一切皆文件

    Linux 本身也是一个基于文件形式表示的操作系统

    Linux目录结构

    • Bin：全称binary，含义是二进制。该目录中存储的都是一些二进制文件，文件都是可以被运行的
    • Dev：该目录中主要存放的是外接设备，例如盘、其他的光盘等。在其中的外接设备是不能直接被使用的，需要挂载（类似windows下的分配盘符）
    • Etc：该目录主要存储一些配置文件
    • Home：表示“家”，表示除了root用户以外其他用户的家目录，类似于windows下的User/用户目录
    • Proc：process，表示进程，该目录中存储的是Linux运行时候的进程。
    • Root：该目录是root用户自己的家目录。
    • Sbin：全称super binary，该目录也是存储一些可以被执行的二进制文件，但是必须得有super权限的用户才能执行。
    • Tmp：表示“临时”的，当系统运行时候产生的临时文件会在这个目录存着。
    • Usr：存放的是用户自己安装的软件。类似于windows下的program files。
    • Var：存放的程序/系统的日志文件的目录。
    • Mnt：当外接设备需要挂载的时候，就需要挂载到mnt目录下

    基础命令

    • ls（list）

      • 列出当前工作目录下的所有文件/文件夹的名称–常用 ls 、ll、ls-a

    • pwd

      • 打印当前所在目录

    • cd

      • 切换工作目录—cd ~ 切换到home目录下

    • mkdir

      • 创建新目录（文件夹）
      • 当一次性创建多层不存在目录时 mkdir -p

    • touch

      • 创建文件

    • cp(copy)

      • 复制文件 #cp 被复制的文档路径 文档被复制到的路径

      • 当复制文件夹时 cp -r 文件夹1 文件夹2

    • mv(move)

      • 移动文档 ：**#mv 需要移动的文档路径 需要保存的位置路径**

    • rm(remove)

      • 删除文档 #rm -rf rm -r linux* (*为通配符)
      • ​ -f：force，强制删除，不提示是否删除
      • ​ -r：表示递归

    • vim

      • 一款文本编辑器
      • 退出！！！！： shift+英文冒号，输入q

    • 输出重定位>/>>

      • #正常执行的指令 > / >> 文件的路径

      • 文件可以不存在，不存在则新建

      • >：覆盖输出，会覆盖掉原先的文件内容

        >>：追加输出，不会覆盖原始文件内容，会在原始内容末尾继续添加

    • cat 打开

      • 直接打开文件cat 文件的路径
      • cat有直接打开一个文件的功能。

    • cat合并

      • 可以对文件进行合并
      • #cat 待合并的文件路径1 待合并的文件路径2 …. 文件路径n > 合并之后的文件路径

    Vim基本模式切换命令

    • 进入插入模式
      • i：在光标当前位置插入文本。例如，当光标在一个字符之前，按下i后，输入的文字会插入到这个字符之前。
      • a：在光标当前位置的下一个字符处开始插入文本。比如光标在某个字符上，按a后，输入的内容会插入到该字符之后。
      • o：在光标所在行的下一行插入新的一行并进入插入模式。这对于在文档中间添加新的段落等操作很方便。
    • 返回命令模式（普通模式）
    • 按下Esc键可以从插入模式返回命令模式。这是最基本的操作，因为很多命令都需要在命令模式下执行。
    1. 光标移动命令 - 基本移动
    • h：将光标向左移动一个字符。
    • j：将光标向下移动一行。
    • k：将光标向上移动一行。
    • l：将光标向右移动一个字符。
    1. 以单词为单位移动
      • w：将光标移动到下一个单词的开头。例如，在一个包含多个单词的句子中，按w可以快速在单词之间跳转。
      • b：将光标移动到上一个单词的开头。如果想回到前面的单词，就可以使用b命令。
      • e：将光标移动到下一个单词的结尾。这在需要编辑单词的末尾部分时很有用。
    2. 行内快速移动
    • 0（数字零）：将光标移动到当前行的开头。不管光标在当前行的什么位置，按0后，光标会迅速跳到行首。
    • $：将光标移动到当前行的结尾。对于在行尾添加内容等操作，使用$很方便。
    1. 页面移动
    • Ctrl + f：向下翻一页（f代表“forward”）。当文档较长时，可以用这个命令快速浏览后面的内容。
    • Ctrl + b：向上翻一页（b代表“backward”）。用于查看前面的页面内容。

    文本编辑命令

    1. 删除操作
    • x：删除光标所在位置的字符。如果发现某个字符输入错误，直接按x就可以删除它。
    • dw：删除从光标位置开始到下一个单词开头的内容。比如想删除一个单词，将光标放在单词的开头，然后按dw即可。
    • dd：删除光标所在的整行。这在删除不需要的行时非常高效。
    1. 复制和粘贴操作
    • yy：复制光标所在的整行。复制后的内容可以在其他位置粘贴。
    • p：将复制或删除的内容粘贴到光标之后。如果是复制了一行，按p后，这一行会粘贴到光标所在行的下一行。
    • P（大写）：将复制或删除的内容粘贴到光标之前。与p的粘贴位置相反。

    文件操作命令

    1. 打开文件
      • vim filename：在命令行中输入这个命令来打开一个名为filename的文件。如果文件不存在，会创建一个新的空文件。
    2. 保存文件 - 在命令模式下，输入:w，然后按下回车键，就可以保存当前文件。如果是新建的文件，需要指定文件名来保存，可以使用:w newfilename。
    3. 退出Vim
    • :q：如果文件没有修改，这个命令可以退出Vim。
    • :q!：强制退出Vim，不保存文件的修改。这在你对文件进行了一些不需要保存的操作，比如只是查看文件内容，想要快速退出时很有用。
    • :wq：保存文件并退出Vim。这是比较常用的操作，在完成文件编辑后，用这个命令可以安全地退出并保存修改。

    进阶指令

    df

    • 查看磁盘的空间
    • #df -h -h表示以可读性较高的形式展示大小

    free

    • 查看内存使用情况
    • #free -m -m表示以mb为单位查看
    • Swap：用于临时内存

    head

    • 查看一个文件的前n行，如果不指定n，则默认显示前10行
    • #head -n 文件路径 【n表示数字】

    tail

    • 查看一个文件的未n行，如果n不指定默认显示后10行
    • #tail -n 文件的路径 【n表示数字】
    • 通过tail指令来查看一个文件的动态变化内容【变化的内容不能是用户手动增加的】
    • #tail -f 文件路径（该命令一般用于查看系统的日志比较多）

    less

    • 查看文件，以较少的内容进行输出，按下辅助功能键（数字+回车、空格键+上下方向键）查看更多

    • #less 需要查看的文件路径（退出时按q）

    wc

    • 统计文件内容信息（包含行数、单词数、字节数）

    • ​ # wc -lwc 需要统计的文件路径

      ​ # -l：表示lines，行数

      ​ # -w：表示words，单词数 依照空格来判断单词数量

      ​ # -c：表示bytes，字节数

    date(重要)

    • 表示操作时间日期（读取、设置）

    • #date 2018年 3月 24日 星期六 15:54:28
      #date +%F 2018-03-24
      #date “+%F %T” 2018-03-24 16:01:00
      #date “+%Y-%m-%d %H:%M:%S” 2018-03-24 16:01:00
      #date -d “-1 day” “+%Y-%m-%d %H:%M:%S” 获取之前或者之后的某个时间（备份）

      符号的可选值：+（之后） 或者 - （之前）

      %F：表示完整的年月日

      %T：表示完整的时分秒

      %Y：表示四位年份

      %m：表示两位月份（带前导0）

      %d：表示日期（带前导0）

      %H：表示小时（带前导0）

      %M：表示分钟（带前导0）

      %S：表示秒数（带前导0）

    cal

    • 用来操作日历的

    • #cal 等价于 #cal -1 直接输出当前月份的日历
      #cal -3 表示输出上一个月+本月+下个月的日历
      #cal -y 年份 表示输出某一个年份的日历

    clear/ctrl + L

    • 清除终端中已经存在的命令和结果（信息）

    管道(|)

    • 管道符：|

    • 管道一般可以用于“过滤”，“特殊”，“扩展处理”

    • 管道不能单独使用，必须需要配合前面所讲的一些指令来一起使用，其作用主要是辅助作用。

    • 过滤案例：需要通过管道查询出根目录下包含“y”字母的文档名称 #ls / | grep y
      特殊用法：通过管道的操作方法来实现less的等价效果**** #cat 路径|less
      扩展处理：统计某个目录下的文档的总个数 #ls / | wc -l

    高级指令

    whoami

    • 我是谁

    id

    • 查看用户基本信息

    hostname

    • 操作服务器主机名

    ps -ef

    • 主要是查看服务器的进程信息（配和管道符、grep 过滤使用）

    • -e：等价于“-A”，表示列出全部的进程

    • -f：显示全部的列（显示全字段）

    • 字段含义

      • UID：该进程执行的用户id；
      • PID：进程id；
      • PPID：该进程的父级进程id，如果无父级进程，则为僵尸进程（parent process ID）；
      • C：Cpu的占用率，其形式是百分数；
      • STIME：进行的启动时间；
      • TTY：终端设备，发起该进程的设备识别符号，如果显示“?”则表示该进程并不是由终端设备发起；
      • TIME：进程的执行时间；
      • CMD：该进程的名称或者对应的路径；

    top

    • 查看服务器的进程占的资源

    • 进入命令：#top （动态显示）

      退出命令：按下q键

    • 表头含义

      • PID：进程id；

      • USER：该进程对应的用户；

      • PR：优先级；

      • VIRT：虚拟内存；

      • RES：常驻内存；

      • SHR：共享内存；

        ​ 计算一个进程实际使用的内存 = 常驻内存（RES）- 共享内存（SHR）

      • S：表示进程的状态status（sleeping，其中S表示睡眠，R表示运行）；

      • %CPU：表示CPU的占用百分比；

      • %MEM：表示内存的占用百分比；

      • TIME+：执行的时间；

      • COMMAND：进程的名称或者路径；

    du -sh

    • 查看目录的真实大小

    • -s：summaries，只显示汇总的大小

      -h：表示以高可读性的形式进行显示

    find

    • 用于查找文件（参数众多，现用现百度即可）
      • -name：按照文档名称进行搜索（支持模糊搜索）
      • -type：按照文档的类型进行搜索
      • **文档类型：“-**”表示文件（在使用find的时候需要用f来替换），“d”表示文件夹

    service

    • 用于控制一些软件的服务启动/停止/重启
    • #service 服务名 start/stop/restart

    kill

    • 表示杀死进程
    • #kill 进程PID（ps 查看PID）
    • #killall 进程名称

    ifconfig

    • 用于操作网卡相关的指令

    reboot

    • 重启计算机
    • #reboot -w 模拟重启，但是不重启（只写关机与开机的日志信息）

    shutdown

    • 关机（慎用）
    • poweroff 、halt、init 0

    uptime

    • 输出计算机的持续在线时间

    uname

    • 获取计算机操作系统相关信息
    • #uname 获取操作系统的类型
    • #uname -a all****，表示获取全部的系统信息（类型、全部主机名、内核版本、发布时间、开源计划）

    netstat -tnlp

    • 查看网络连接

    • ​ -t：表示只列出tcp协议的连接；

      ​ -n：表示将地址从字母组合转化成ip地址，将协议转化成端口号来显示；

      ​ -l：表示过滤出“state（状态）”列中其值为LISTEN（监听）的连接；

      ​ -p：表示显示发起连接的进程pid和进程名称；

    man

    • manual，手册（包含了Linux中全部命令手册，英文）（退出按下q键）
    • #man cp #man命令查询cp指令的用法

    chkconfig –list

    • 开机启动项查询

    • -add 增加

    • -del 删除

    • -level 设置服务在某个级别下开机启动/不启动
